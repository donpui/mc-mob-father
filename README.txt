
# Father Mob mod

Based on skeleton, throws snowball and encourage to go do homework and attack with the book

## Basic commands

- export JAVA_HOME=/opt/homebrew/Cellar/openjdk@17/17.0.13/libexec/openjdk.jdk/Contents/Home
- build project: `./gradlew clean build`
- run in client: `./gradlew runClient`
- summon in MC: ``/summon fathermob:father`
